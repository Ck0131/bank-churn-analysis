## 📌 Project Overview

This project analyzes **bank customer churn** using Power BI to understand why customers leave and identify key patterns across demographics, geography, credit scores, and account behavior.

---

## 🔍 Key Insights

| Metric | Value |
|---|---|
| Total Customers | 10,000 |
| Churn Rate | 20.37% |
| Total Balance | ₹765 Million |
| Average Credit Score | 650.53 |
| Active Members | 48.49% |
| Credit Card Owners | 70.55% |

- 📍 **Geography**: France has the highest customer count, followed by Germany and Spain
- 👥 **Gender**: Slightly more female customers (54.57%) than male (45.43%)
- 💳 **Credit Card**: Majority of customers (70.55%) own a credit card
- ⏳ **Tenure**: Average tenure is 2K with a minimum of 5.51K recorded
